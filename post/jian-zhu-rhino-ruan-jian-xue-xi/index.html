【建筑】rhino软件学习

  1. 2022-09-05
  2. 505字
  3. 2 min read

相关链接：

软件学习：

摘要：快捷键/命令/工具列面板+属性+中键/视图+工作平面+网格+选取

一、快捷键：

F10 打开控制点
F11 关闭控制点

二、命令：

1.线

命令 属性
WIRECUT 线切割体
IT(INSERT) 相交
DF(Dupfaceborder) 复制边框线
AW(Cplane的obiect）-rec-AQ（Cplane的world） 工作平面上画线

2.线

命令 属性
extractisocurve 抽离结构线
extractpipedcurve 抽离结构线
tweencurve 均分过渡曲线
BC（blendcrv） 混接曲线
closecv 闭合开放的曲线
divide 均分
接缝-改接缝点 改曲线周期性

3.线-面

命令 属性
SW1（sweep1） 单轨扫
SW2（sweep2） 双轨扫
loft 多条线
AS(PlanarSrf) 平面线构面
userxtrusion 挤出曲面设置

4.其余

命令
pl命令下
shift+
<度数
@@坐标
ctrl再捕捉点可以垂直或oneview
tab捕捉方向上的路径
tab按一次锁定方向按两次锁定距离
按ctrl移到下面点的捕捉下选择曲面绘制

三、

1.工具列面板：

命令 属性
中键 ctrl工具拖到中键
ctrl+工具 复制
shift+工具 编辑

2.属性-图层+材质+注释：

命令 属性
layerstatemanage 图层状态
ma（matchproperties） 搭配属性
printdisplay 打印线宽
showzbuffer 全黑背景
downloadlibrarytexure 载入材质
text 文字
text-fx-面积 变量面积显示
options-自定义 赋予信息
getusertext 显示

四、

1.视图：

命令 属性
zoom/z+空格 框选
ctrl+右键 放大缩小
F6 当前视窗摄像机
alt+ctrl+右键 固定摄像机转视角
alt+shift+右键 固定摄像机旋转角度
快照 可记录阴影变化与物体变化
clippingplane 截图
enable/disable 激活截面
小技巧 建立 -对齐正对曲面 对齐斜面
F7 格点显示
oneview 格点随旋转变动

2. 工作平面：

cplane

3. 选取：

命令
seldim
selv
selu
selpt
sellast
CTRL+SHIFT+e
ZS
ZE

4. 网格操作

命令 属性
属性自定义网格 nurbs转网格精度高点
导出su格式 可先导出3ds
